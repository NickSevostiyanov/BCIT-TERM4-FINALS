## Course Overview (Topics Covered)

**Foundation (Weeks 1–4):**
- Python data tools: NumPy arrays, Pandas DataFrames, Jupyter Notebook
- Data types, shapes, indexing (iloc, loc), slicing
- Missing value imputation: mean/median/mode, random sample, KNN imputation
- Data cleaning: duplicates, garbage detection (GIGO principle)

**Feature Engineering (Weeks 5–6):**
- Constant and quasi-constant feature removal
- Correlation-based feature selection
- Chi-square test for categorical features
- Mutual information (regression + classification)

**Visualization (Week 7):**
- EDA with matplotlib, Tableau Public

**Machine Learning — Supervised (Weeks 8–11):**
- Regression: Linear regression, evaluation (MAE, MSE, RMSE, R²)
- Classification math: accuracy, precision, recall, F1, confusion matrix
- kNN: distance metrics, choosing k, curse of dimensionality
- SVM: hyperplane, kernel trick, soft margin
- Decision Trees: entropy, information gain, Gini impurity, overfitting
- Random Forest: bagging, feature importance, ensemble
- Model Tuning: GridSearchCV, cross-validation, train/test split
- Imbalanced Data: oversampling (SMOTE), undersampling, class weights

**Unsupervised Learning (Week 12):**
- K-Means: centroids, elbow method, inertia
- Hierarchical Clustering: dendrograms, linkage methods
